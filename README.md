# BFR(Batch File Rename)
批量文件重命名工具

# 开发环境

Windows 10 + Visual Studio 2013 + .NetFramework 2.0

# 功能

* 批量替换文件名中内容
* 按序号批量重命名
* 扩展名字母大小写转换
* 修改文件的创建时间和访问时间
